# INCOLSAN — Landing Page Oficial

Sitio web oficial de **Inversiones Colón Santa María S.A.S. (INCOLSAN)**, proveedor de internet de fibra óptica en Acandí, Chocó, Colombia.

## 🌊 Tecnologías
- React + Vite
- Tailwind CSS
- Lucide React
- Cloudflare Workers (proxy seguro para IA)
- Groq API con Llama 3.1 (chatbot Caná)

## 🤖 Chatbot Caná
Asistente virtual impulsado por IA integrado en la página. Responde preguntas sobre planes, cobertura y soporte. La API key está protegida mediante un Cloudflare Worker como proxy seguro.

## 🚀 Correr localmente
```bash
npm install
npm run dev
```

## 📡 Sobre INCOLSAN
Empresa local comprometida con conectar las comunidades afrodescendientes de la costa del Chocó.
