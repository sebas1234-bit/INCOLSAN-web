# INCOLSAN — Landing Page Oficial

Sitio web oficial de **Inversiones Colón Santa María S.A.S. (INCOLSAN)**, proveedor de internet de fibra óptica en Acandí, Chocó, Colombia.

## 🌊 Tecnologías
- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## 🚀 Correr localmente
```bash
npm install
npm run dev
```

## 📡 Sobre INCOLSAN
Empresa local comprometida con conectar las comunidades afrodescendientes de la costa del Chocó.

**"Tu vida mejora con nuestra conexión"**
